# 📊 Recruitment Analysis Dashboard | Power BI

## Project Overview
This project is an interactive Recruitment Analysis Dashboard built using Power BI with Microsoft Excel as the data source. It provides meaningful insights into the recruitment process through KPIs, charts, slicers, and interactive visualizations. The dashboard helps monitor candidate details, hiring status, locations, domains, and salary trends.

##  Features
- Total Candidates KPI
- Selected & Pending Candidates
- Average CTC (LPA)
- Average Experience
- Location-wise Candidate Analysis
- Domain-wise Candidate Distribution
- Selection Status Analysis
- Interactive Slicers & Filters
- Detailed Candidate Information Table
- Dynamic Data Refresh

## 🔄 Dynamic Refresh Demo
This project includes a demo video demonstrating Power BI's dynamic refresh capability.

In the demo:
- The candidate's location was changed from *Delhi* to *Noida* in the Excel source file.
- After refreshing the dataset in Power BI, the dashboard updated automatically.
- The location-wise analysis reflected the updated data, showing how Power BI stays synchronized with the connected Excel file.
